# ЕМ СИ ЕМ ЕООД — Уеб сайт

Кодовото хранилище съдържа статичната уебстраница на "ЕМ СИ ЕМ ЕООД" — фирма за професионални счетоводни услуги в Костинброд.

## Съдържание

- `index.html` — основната страницa (hero, услуги, контакти и т.н.)
- изображения и ресурси (hero.webp, msmlogo.png, service*.webp и др.)
- `README.md` — този файл

## Локален преглед

От корена на проекта стартирайте прост HTTP сървър и отворете браузър на `http://localhost:8000`:

```bash
python -m http.server 8000
# или (PowerShell)
# Start-Process "http://localhost:8000"
```

## Бележки за разработчици

- JSON-LD: В `index.html` е добавен Schema.org JSON-LD за локалния бизнес (`AccountingService`) и `WebPage` с поле `creator`.
- `creator` е представен като организация с `@id`: `https://moverstudio.online/#organization`.

## Лиценз и собственост

Проектът и съдържанието на сайта са собственост на "ЕМ СИ ЕМ ЕООД". За въпроси относно използване или публикуване на съдържание, моля свържете се с притежателя на сайта.
